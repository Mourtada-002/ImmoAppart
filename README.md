# 🏠 ImmoAppart

**ImmoAppart** est une plateforme moderne de location de biens immobiliers, conçue pour faciliter la recherche, la gestion et la réservation de logements pour des séjours courts ou longs.

---

## 📌 Description

ImmoAppart propose une large gamme de biens immobiliers, incluant :

- 🏡 Maisons résidentielles  
- 🌴 Locations de vacances  

Les utilisateurs peuvent louer des logements pour :

- ⏳ Courts séjours  
- 📅 Longs séjours  

La plateforme intègre également des services complets pour garantir une expérience fluide et professionnelle.

---

## ⚙️ Fonctionnalités principales

- 🔍 Recherche de biens immobiliers
- 📄 Consultation des détails des logements
- 📆 Réservation de séjours
- 💬 Communication entre clients et propriétaires
- 🛠️ Gestion immobilière intégrée
- 🧹 Suivi de l’entretien des biens
- 🎧 Assistance client

---

## 🛠️ Technologies utilisées

- Flutter (Mobile App)
- Firebase (Backend)
- HTML / CSS / JavaScript (Web)
- Autres outils en cours d’intégration
